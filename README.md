## 🐾 Release Mascot

<!-- MASCOT -->
![Release Mascot](mascots/1.0.0.png)
<!-- /MASCOT -->




A GitHub Action that generates a unique mascot for every release.

## Usage

```yaml
- uses: OwainWilliams/OC.ReleaseMascot@main
  with:
    Tag: ${{ github.event.release.tag_name }}
```
